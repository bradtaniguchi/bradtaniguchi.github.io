health-portal-app

This project is closed source and created for work. It consisted of a large scale Angular application with a TypeScript back-end that ran in GCP.

This project included a large number of features including:

  • biometric lab integration
  • COVID vaccination tracking for multiple states
  • Custom API integration
  • Administration
  • Data importing
  • Data exporting
